We analyze US gun murder data collected by the FBI.
Esto es parte del curso de EdX, con ello estoy aprendiendo a generar repositorios en GitHub.
Además de utilizar las principales funcionalidades de esta herramienta de productividad.

download-data.R - download csv file to a data directory

wrangle-data.R - creates a derived dataset and saves as R object in rda directory

analysis.R - generates a plot and saves it to the figs directory
